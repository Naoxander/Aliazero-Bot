Home / Start -> Inicio

Accion:
Muestra Menu de Inicio.

Ejemplo:
~Home / ~Start

Menus / Mp -> Lista de Menus

Accion:
Muestra todos los menus que esten disponibles.

Ejemplo de Uso:
~Menus / ~Mp

Sticker / s -> Menu de Stickers

Accion:
Crea stickers a partir de imagenes.(Solo imagenes)

Ejemplo de Uso:
~s / ~Sticker [la imagen]

Animated / sa -> Menu de Stickers

Accion:
Crea stickers a partir de videos o gif.
Recomendacion: La duracion maxima que no exceda los 10 segundos.

Ejemplo de Uso:
~sa / ~Animated [video/gif]

Img / Jpg -> Menu de Stickers

Accion:
Transforma un sticker en imagen.

Ejemplo de Uso:
~img / ~jpg [sticker]

gg-imagen / gg-img -> Menu de Interes

Accion:
Recibe resultados de Google a partir de la Imagen que brindes.

Ejemplo de Uso:
~gg-img / ~gg-imagen [imagen]

Math / Cal / Calcular -> Menu de Interes

Accion:
Calculadora Basica y Cientifica.
Recomendacion:
si quieres usar la calculadora en modo experto, lee el manual de codigos con el comando ~Tutomath.

~Math / ~Cal / ~Calcular 10+3

Wiki -> Menu de Interes

Accion:

Ejemplo de Uso:
~Wiki covid-19

Yplay -> Menu de Youtube
